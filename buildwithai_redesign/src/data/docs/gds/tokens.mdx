# Design Tokens

The ODBWAIL Design System uses tokens for colors, typography, spacing, radii, shadows, and motion.

## Categories

- Color tokens
- Typography tokens
- Spacing tokens
- Radius tokens
- Shadow tokens
- Motion tokens

## Usage

Tokens are defined in `src/assets/gds.css` and applied globally.
# GDS Tokens

This document lists the core design tokens used across the platform.

- Colors: `--color-primary`, `--color-accent`, `--color-neutral-*`
- Typography: `--font-family-headings`, `--font-family-body`
- Spacing: `--space-1` .. `--space-12`
