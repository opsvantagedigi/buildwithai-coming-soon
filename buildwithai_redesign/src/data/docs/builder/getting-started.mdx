# Getting Started with the AI Website Builder

The Build With AI Website Builder helps you generate a complete, multi‑page website using AI. This guide walks you through the first steps.

## Steps

1. Describe your business or project.
2. Choose a domain (powered by Domain‑Intel).
3. Select design preferences.
4. Generate your site.
5. Edit sections using AI or manual controls.
6. Deploy your site.
