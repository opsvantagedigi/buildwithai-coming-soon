# RDAP Details

RDAP replaces WHOIS and provides structured, machine‑readable domain data.

## What We Normalize

- Domain status
- Registrar
- Nameservers
- Expiry date
- Ownership flags
- Abuse contacts

## Integration with Builder

RDAP data is cached and used to:

- Recommend domains
- Validate user‑provided domains
- Display domain details in the builder
