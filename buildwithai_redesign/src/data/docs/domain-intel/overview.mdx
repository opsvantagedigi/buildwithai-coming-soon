# Domain Intelligence Overview

Domain‑Intel powers domain availability, pricing, RDAP details, and DNS insights inside the AI Website Builder.

## Capabilities

- Domain availability checks
- RDAP canonicalization
- Pricing lookup
- DNS record viewer
- KV caching with Upstash fallback
- Warm job for popular domains

## Why It Matters

Domain‑Intel makes the builder feel “alive” by validating domain ideas in real time.
